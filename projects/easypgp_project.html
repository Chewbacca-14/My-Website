EasyPGP

Android & IOS

The application allows decrypting messages that were encrypted using PGP encryption. All you need is the password and the PGP private key from the key pair used to encrypt the message. Messages can be decrypted in two ways: by pasting the encrypted message or by scanning the QR code containing the encrypted message.

If you don't have PGP keys, you can generate them directly in the application, choosing from three key lengths: 2048 bits, 4096 bits, and 8192 bits. After gener ating the keys, they will be stored in a secure storage on your phone. And if you already have your key pair, you can upload them to the application. The application also offers you the option to secure access using a password, fingerprint, or Face ID.

Main Technologies

  • Dart
  • Secure Storage
  • OpenPGP
