Commercial App - Still in Development

Papi Burgers

Android & IOS

This application serves as a delivery service for a project that owns various restaurants. At the outset, users select the restaurant from which they want to order food, add items to their cart, and proceed to checkout. The application features integrated payment functionality.

Additionally, an Admin application has been developed for comprehensive control over the entire platform. Within the admin application, administrators can edit/add/delete restaurants, menus, restaurant locations on the map, post job vacancies, and much more.

Main Technologies

  • Flutter
  • Dart
  • Firebase
  • Google Maps
  • Yookassa
